# Digital Literacy Portfolio  

## Project Overview  

This Digital Literacy Portfolio has been created as part of the CSE0001 – Digital Literacy course at VIT Bhopal University. The aim of this project is to develop a clear understanding of digital tools, responsible online behavior, professional communication, and cyber safety.  

Through this portfolio, I have completed multiple practical tasks that demonstrate my ability to use digital platforms effectively, maintain a professional online presence, and understand real-world cyber threats. Each task is organized into separate folders for clarity and easy access.
